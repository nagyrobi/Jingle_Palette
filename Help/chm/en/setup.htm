System requirements (recommended values, these are not mandatory):

  • PC with Windows compatible soundcard with WDM drivers

  • Touch Screen for easy usage. If not, a mouse will be as good :)

  • True Color graphic display, at least at 800x600 resolution

  • Ultra DMA Mode 6 IDE, or SCSI Hard Drive to hold the jingles in Wave or Mp3 format

  • 1 GHz Processor clock should be enough. Tested only with 2000 MHz until now...

  • Microsoft Windows operating system, version 98SE, Me, 2000, XP

  • at least 64 MB of RAM

  • DirectX 8

Jingle Palette comes with an installation package packed in the setup_jingle_palette.zip file that you can download from the website. It always contains the latest version of the program. You can use your favorite archive manager program (like WinZip or WinRAR) to unzip it to a folder of your choice.

Please note that you need administrative privileges on a Windows 2000 or XP machine to install Jingle Palette.

Installation begins by double clicking setup_jingle_palette.exe. It is recommended to close all the other windows that you have open, including other programs that play music, edit documents, etc. Clicking on Next will bring up the window where you can select the installation folder. It's best to leave the default value. Afterwards, the installer unpacks the program and system files to the destination folders, registers them and makes all the necessary steps to prepare your system for usage with Jingle Palette. At the end of the installation you may have to restart your computer to finalize the installation process. Please do so before using Jingle Palette.

Pressing Finish will end the installation process. You can start the program by double-clicking its icon from the desktop, or from the Start menu.
