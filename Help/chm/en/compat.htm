Jingle Palette can work very well along other audio players, including Winamp or other professional broadcasting automation tools. It totally depends on the other programs if they can tolerate or not external players in the same time they play.

Note that many software manufacturers state that using other players among  their own may damage the job that they do. Sometimes it's true, but sometimes it's not.

If you have a system that has one or more soundcards, and DirectX is installed, it's 99% certain that you can use Jingle Palette along with your existing software. Especially if the other software is also working with DirectX, then it's 100% sure that they can work together without bothering each other, even on the same soundcard for broadcasting output.

Check the other software's documentation. If it states that "DirectX is required" or during its installation it warns you about that, or installs DirectX, etc, then its almost sure that it's using it. Then, there's no problem. (You can check the version of your installed DirectX by typing "dxdiag" in the Run box from the Start Menu)

There are lots of possible setups. You can use one single soundcard for the output in the main studio mixer, and use Jingle Palette along with some music player/broadcast manager. You can use two soundcards like some broadcast manager software require, the first one for the ON AIR output, and the second one for monitoring, in this case you must setup Jingle Palette to output on the first card (check Settings section).

You can even throw in an extra soundcard if you want to, only for Jingle Palette, that you can also wire in the broadcast mixer on a separate channel. This in case of some player/broadcast manager that doesn't use DirectX or likes to lock the soundcards exclusively for itself; or if you want to be 1000% sure that they run  independently.

Windows 2000 or Windows XP is HIGHLY recommended. They do multitasking way better and multiple applications run in parallel much better and more stable. This is not a warranty though that the two or more programs will run together smoothly. Memory, running services, networking, especially the quality of the hardware components, all affect PC performance. Jingle Palette offers you the possibility to choose the process thread priority if you need to fine tune compatibility between programs.

It's best to try yourself, but watch out, you're on your own responsibility.

Make sure you have backups of your broadcast and data before you start experimenting. Never make tries while you're ON AIR! It's best to try first on a separate computer, the production studio PC or the newsroom for example.

Jingle Palette plays MP3, MP2, MP1, MPA, OGG and WAV audio files or streams. It also supports compressed WAV files with the condition that you have installed the appropriate codec on your system.
