Settings allow you to customize Jingle Palette the way you like.

Setting the Volume

You can set the volume level globally, for the general output of Jingle Palette, and individually, for each jingle. This individual setting is saved in the palette.

To set the individual volume for each jingle, click Assign new Volume level to Jingle button. All the buttons on the palette will turn blue, so you can choose a jingle to set its volume.

After clicking the desired jingle button, on the settings panel the blue volume fader will appear, allowing you to precisely set the volume by dragging it. Right-clicking on a jingle button and selecting Assign Volume Level will bring up the same thing:

After setting the desired volume, click on Save Volume to save it in the palette and return to the initial state.

Note: You can start playing the jingle and after that press the Assign new Volume level to Jingle button to set the volume while listening.

To set the main output volume level from Jingle Palette, click Settings... and in the configuration window, on the Audio tab, drag the red fader to select the desired volume.

Note: You can start playing a jingle and after that go to Settings to set the main output volume while listening. To reset to 0 dB, you can click the number above the fader.

Setting Loop mode individually

Each jingle can be played in Loop mode, and this preference is also saved in the palette. To do that, click the Assign/Clear Loop Mode button. All the palette will turn orange, allowing you to select a jingle to be played in Loop mode:

Clicking a jingle will enable its Loop playback mode. To remind you about this state, an orange LED is lit up on the button. If you play a jingle while this is on, that jingle will be looped (e.g. repeated) until you stop it in some way, either with STOP ALL, or by clicking it again without the Autorepeat mode on.

Note: For best results, it is recommended to store jingles that you wish to loop in PCM WAV, Mp1 or Mp2 format only. We do not recommend Mp3 for looping.

Right-clicking on a jingle button and selecting Set(Clear) Loop Mode will to the same.

Advanced Settings

Clicking the Settings... button will bring you up the Configuration Settings window, which will allow you to choose different options for Jingle Palette:

Here you can choose the Time and Date display format, the number of seconds after which the warning rectangle will appear on the display, the color of it.

The Auto Mix time can be set in milliseconds. The default is 1500 (1.5 seconds) but you can specify any value between 100 and 3000.

Note: The values of the Remaining warning time, Auto Mix time, etc. can be changed by clicking the little Up/Down arrows, or from keyboard by pressing UP, DOWN keys in the corresponding number box. Pressing LEFT or RIGHT will change the values more rapidly.

The Time Announce signal can be chosen here in a similar way that jingle assignment is done.

The output device can also be chosen from the settings window. If you change the output device, all output will stop, the audio module will reset and reload the palette on the new device, as the message box will warn you. So if you have a jingle on air, you can wait until it finishes if you want and then press OK:

If you select an unsupported device, you will be warned accordingly.

To set the main output volume from this window, see above.

The Process Priority Thread can be chosen here from Idle to Time Critical. This is useful in case you want to fine tune the software compatibility between programs running together on the same computer system. Multitasking environments differ from one operating system to another. In any case, the Time Critical priority is not recommended, it can cause system lockup and instability. A message will warn you in this case to be cautious. Save all errors to "Error.log" file checked saves not only the program errors, but also the missing jingles, and other user-generated errors to a logfile to check later. Always On Top of all the other windows checked makes Jingle Palette to be the topmost window on your desktop. That's helpful when using with other broadcasting tools.

Changing the language of the interface can be done easily, by selecting the desired language from the dropdown box. To have Jingle Palette translated in your language, see Translating Jingle Palette in other languages section. It's easy and anyone can do it.

You can choose to save the program errors to a log file. This is useful if a certain error occurs repeatedly, because you can send it to the author for debugging and problem fixing.
