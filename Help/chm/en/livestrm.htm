Remote streams can be played through internet or intranet with Jingle Palette. The supported formats for the streams are MP3, MP2, MP1, MPA, OGG or WAV audio, respectively live Shoutcast / Icecast streams.

On the Play Stream tab, you can type the URL of the remote file in the bar. Clicking the Play Remote Location button will start the connection with the server and play the file. In case of some error, the corresponding message will appear in the status box.

Auto Mix by Time Announce function is useful if you want to stop the playback of the remote stream with the Time Announce. Simply set up Time Announce at the desired moment and then click this button.

The address bar remembers all your URLs that you played in the history, so you can easily reuse them and don't have to type them in again.

Note: If your internet/intranet connection is lost during playback, it will be stopped automatically after the remaining sound from the download memory buffer will empty.

Warning! The remote location playback will be stopped when STOP ALL button is pressed.
