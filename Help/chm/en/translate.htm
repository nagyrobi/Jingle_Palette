Translating Jingle Palette in your own language

If you cannot find your language in the language dropdown box from the Configuration Settings window, you can make your own translation.

Localized strings can be found in the "language.ini" file, located in the installation folder of Jingle Palette (usually C:\Program Files\Jingle Palette). Double-clicking this file will open it in Notepad for editing.

The file follows the structure of a standard ini file, in the following way:

[Section]

Enrty1=Value

Entry2=Value

In our case, the section name is the name of the language and the entries are the localized strings for the defined controls, like this:

[English]

mb_tmAnJin=Select jingle to be announced...
mbtAbout=About...
mbtDn=Next Palette
mbtExit=Exit

To add your own language translation to Jingle Palette, do the following:

  • create a backup copy of "language.ini". (Right-click and drag in the same folder and release, select "Copy Here" and a file called "Copy of language.ini" will be created. This will be useful if you mess up the original "language.ini" by mistake, to restore it.)
  • exit Jingle Palette
  • open "language.ini" for editing in notepad
  • select with the cursor the entire "English" section, including all the entries until the next language
  • copy it and paste it at the end of the file
  • change the name of the section to your language
  • begin translating the strings that follow the equal "=" sign. Do NOT modify anything before the "=", or else your translation will not work!
  • save "language.ini"
  • run Jingle Palette, go to settings and select your language.

At the end you should have something like this:

[Your Language]

mb_tmAnJin=you translated it...
mbtAbout=you translated it...
mbtDn=you translated it
mbtExit=you translated it

After you fully finalized your translation, please send a copy to the author so he can share it with the other users in the next release. Thanks!
